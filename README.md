# OpenGL2D-Project

# Intro
This a small 2D project using OpenGL, C++, and glut.h.

- - - -
## Controls
Key | Action|
------------- | -------------
<kbd>→</kbd> | Move right
<kbd>←</kbd> | Move left
<kbd>W</kbd> | Camera up
<kbd>S</kbd> | Camera Down
<kbd>↑</kbd>  | Move up
<kbd>↓</kbd>  | Move down

- - - -
## How to run ?
clone the project into Visual Studio and run it. (simply open the .sln file and run it)
